
<h1>Portifolio Dev | Responsivo</h1>
<strong>📜 Descrição</strong><br>
<p>Portifólio para uso pessoal com a finalidade de exibição de projetos e informações profissionais.<br>
Tecnologias utilizadas: <strong>HTML5, CSS3 e JavaScript.</strong><br>
Responsividade para dispositivos móveis do mobile-s (320px) ao desktop 4k (2560px).</p>

<h1>Site responsivo - mobile-s</h1>

![image](https://github.com/user-attachments/assets/875546b0-53ea-4343-acdf-ff1d40289cb2) <p>Site com responsividade, utilizando CSS e JavaScript (Especialmente na barra de navegação) para o redimensionamento. A função do CSS3 Media Query possibilitou fazer o redmimensionamento da barra de navegação para o mobile, sendo assim quando a largura da barra de navegação for menor que 768px (Tamanho Tablet) a barra para desktop é colocada como "display: none" através do media query e não é mais exibida na tela, além disso surge um botão para acessar cada segmento do site, quando clicado.   </p>
